# S02-26-Equipo_42_Data_science_Business_Intelligence_Robotica_Quirurgica
Modelo de Business Intelligence para evaluar la factibilidad de Justina, un robot para cirugía renal mínimamente invasiva en América Latina. Analiza mercado (TAM/SAM/SOM), costos y modelos de negocio, CAPEX/OPEX/TCO, estrategias de adopción y KPIs, usando datos públicos y dashboards para apoyar decisiones estratégicas.
